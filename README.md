# Требования к программе

## Основные требования

-  **Реализовать наследник `std::pmr::memory_resource`**, который реализует стратегию работы с памятью согласно варианта задания

-  **Поддержка переиспользования памяти** - `memory_resource` должен позволять переиспользовать освобождаемую память (освобожденную ранее с помощью `do_deallocate`)

-  **Автоматическая очистка памяти** - `memory_resource` при уничтожении должен подчищать всю неосвобожденную ранее память

-  **Шаблонный контейнер** - реализовать контейнер согласно варианту задания, который использует созданный `memory_resource` через шаблон `std::pmr::polymorphic_allocator`

-  **Итератор** - реализовать итератор к созданному контейнеру. Итератор должен соответствовать `std::forward_iterator_tag`

-  **Демонстрационный код** - создать код, демонстрирующий работу контейнера с простыми (`int`) и сложными типами (`struct` с несколькими полями)
